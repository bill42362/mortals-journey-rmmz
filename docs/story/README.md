## 每份章節文件的結構約定

1. **劇情大綱** — 本章在主線中的定位與事件梗概
2. **章節目標** — 敘事目標 + 教學/系統目標
3. **地圖規劃** — 需要哪些地圖（供人工繪製參考）
4. **地圖事件清單** — 各事件的觸發/行為/實作方式（內嵌地圖事件，或在需複用/掛鉤核心系統時用共通事件）
5. **關鍵對話文本** — 重要橋段台詞
6. **與核心系統的關聯** — 本章觸及哪些系統 / 教學點
