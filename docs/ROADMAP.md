## 後續（demo 之後）

延後的系統與可行性/實作路徑參考（demo 範圍已排除，留待後續逐步加入）：

| 系統 | 可行性 | 實作路徑 |
| --- | :---: | --- |
| 製符 | 🟢 | 與煉丹同屬合成框架（ItemCraftingSys），複用即可 |
| 煉器 | 🟢 | 同合成框架 |
| 法寶戰鬥（含自爆） | 🟡 | 技能/道具表現；「自爆」= 消耗法寶的一次性高傷技能，需自訂消耗與平衡 |
| 仙法戰鬥（深化） | 🟢 | 原生技能系統承載，後期深化 |
| 傀儡戰鬥 | 🟡 | 召喚/臨時隊友機制，需外掛 |
| 屍傀戰鬥夥伴 | 🟡 | 轉化為「召喚出戰的隊友」，與傀儡共用框架 |
| 靈寵培育 | 🟠 | 需自訂養成系統（屬性成長/餵養/出戰），工作量大 |
| 符陣煉製/使用 | 🟠 | 戰前布陣/場地效果偏自訂，複雜度高 |
| ARPG 戰鬥（Action Combat） | 🟡 | 與回合制二擇一，留作後期實驗 |

> 可行性分級：🟢 容易 ／ 🟡 中等 ／ 🟠 困難。
